iOS Popovers made with CSS

Recently, I was building a mobile web application slated for the iOS platform when I came across a problem. There was a native iOS feature that I wanted to use, but couldn't find a way to implement natively in the mobile browser.

The feature in question was the blue popover menu used by native apps to organize menu options and other modular content. The links below are examples of what I'm talking about.

Flipboard     Mobile Safari     Penultimate

Naturally, I turned to my confidant and mentor Google for council, but disappointingly, I was turned away. The closest I could find to the feature in question was the way the html select element is implemented in Safari, but that seemed hackish, and lacked many of the features I was after, like title, navigation, action buttons, etc.

Less than ideal HTML Select Element

Most people would have thrown their hands up and quit, or found a shortcut to accomplish the same goal but with less glossy iOS perfection. Not me, I'm an asshole, and I decided to spend some time building out the feature with javascript and CSS. While there are still many javascript features still in the works, the basic HTML and CSS structure are provided in the source files for you to start playing with. Bust out your iPad or Xcode's iPad simulator to check out the demo below!
